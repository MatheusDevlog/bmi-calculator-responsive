# 🧮 Calculadora de IMC

Projeto de uma calculadora de IMC (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript puro.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox e responsividade)
- JavaScript (manipulação de DOM)

## 📌 Funcionalidades

- Cálculo automático do IMC
- Classificação do resultado (abaixo do peso, ideal, sobrepeso, obesidade)
- Layout totalmente responsivo
- Interface moderna e intuitiva

## 📱 Responsividade

O layout se adapta para:
- Desktop
- Tablet
- Mobile

## 📚 Aprendizados

Neste projeto pratiquei:
- Manipulação de eventos com JavaScript
- Estruturação de layout com Flexbox
- Uso de media queries
- Organização de código front-end
